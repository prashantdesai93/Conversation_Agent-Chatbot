Steps to execute the project as follows,

Dependencies to be installed:
1. Numpy
2. heapq
3. TensorFlow
4. six

Data preparation:
To prepare dataset execute prepare_data.py in data folder
	We have already prepared encoding and decoding data files

To Train data:
1. In seq2seq.ini file, set mode=train
2. Run--> python main.py

To Test:
1. In seq2seq.ini file, set mode=test
2. Run--> python main.py
